# harbor-workbench installer

Install harbor-workbench with

```
curl -fsSL https://raw.githubusercontent.com/Proximal-Labs/hbw-installer/main/setup.sh | bash
```

Experimental version install:

```
curl -fsSL https://raw.githubusercontent.com/Proximal-Labs/hbw-installer/main/setup.sh | WORKBENCH_REF=main bash
```
